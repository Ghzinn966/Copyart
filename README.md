## 🎯 Sobre o Projeto
Este projeto foi desenvolvido como requisito para a disciplina de Projetos em Sistemas. Nosso objetivo é atender a uma demanda real do(a) **CopyArt**, que atua no setor **Gráfico e Comunicação Visual**. 

O sistema visa resolver a falta de informações e detalhamento dos serviços, contato via web, com integração de E-mail via formulário online, WhatsApp via link direto pra conversa, Instagram pra conhecimento geral da empresa e cia; e localização e informações renovadíssimas de último momento pra Gráfica Expressa Copyart.
## ✨ Funcionalidades
- [X] Painel de portfólio + avaliações de clientes satisfeitos
- [X] Automação de envio de e-mails/mensagens via botão linkado com WhatsApp e E-mail com formulário online
- [X] Localização via Google Maps
- [ ] Adição de E-Commerce para fluxo de estoque e produtos (em breve)
